## Objetivo

Este projeto tem como objetivo verificar a correta configuração do ambiente de desenvolvimento da linguagem Dart, além de validar a execução de um programa simples utilizando o terminal e o uso de Git e GitHub para versionamento.

## Como executar o programa

1. Instale o Dart: https://dart.dev/get-dart
2. Clone este repositório:

   ```bash
   git clone https://github.com/MatheusAlexanderAlves/trabalho-dart-hello-world.git
   ```
3. Acesse a pasta do projeto:

   ```bash
   cd trabalho-dart-hello-world
   ```
4. Execute o programa:

   ```bash
   dart run main.dart
   ```
